# UpLog
기업, 개발자, 의뢰인이 소통할 수 있는 릴리즈 노트 공유 시스템을 개발하였습니다. 현재 리팩터링을 진행하고 있습니다. 이 프로젝트는 개발을 시작하기 전에 2주 동안 프로젝트의 컨셉과 UI를 설계하고 개발을 시작했습니다.
